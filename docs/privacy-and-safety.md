# Privacy and safety

NeXa RoVe is being developed as a local-first assistant and robotics project. Because it involves voice, camera, memory, hardware, and testing, this public repository has strict boundaries.

This repository will not include:

- private user data
- private memory files
- raw audio
- raw camera images
- raw transcripts
- logs
- diagnostics
- `.env` files
- tokens
- private source code
- full internal architecture
- private prompts
- private configuration files
- secrets or credentials

Public material should be reviewed before it is added. Screenshots, photos, videos, and notes should not reveal sensitive screens, private conversations, device identifiers, tokens, local paths, or personal data.
