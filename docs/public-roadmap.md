# Public roadmap

This roadmap is deliberately broad. It is intended to show the public direction of the project without exposing private plans or implementation details.

## Near term

- Add safe photos of the hardware setup
- Add short demo clips that do not show private data
- Improve the public project documentation
- Add simple diagrams for the system at a high level
- Keep a public progress log with selected updates
- Add more simplified examples where they are useful

## Later ideas

- Show more of the user interface in a safe way
- Share short write-ups about hardware lessons learned
- Add public notes about local-first AI trade-offs
- Document safe demo workflows
- Add more general hardware and software overview pages

## Long-term direction

- Continue developing NeXa RoVe as a local-first physical assistant
- Improve reliability on real hardware
- Keep privacy and safety as core project constraints
- Share selected progress publicly without exposing the private runtime
