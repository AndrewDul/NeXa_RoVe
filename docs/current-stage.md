# Current stage

NeXa RoVe is an active personal project. The current work is focused on building a stable local-first assistant and robotics foundation rather than presenting a finished product.

Public-safe areas currently being developed include:

- local AI direction
- voice interaction work
- local speech input and output experiments
- UI work for assistant feedback and status
- Raspberry Pi and hardware integration
- sensors and robotics exploration
- display and interface testing
- testing and debugging on real hardware
- safety thinking around physical movement
- learning support and personal assistance ideas

The project is still changing as hardware, software, voice, interface, and safety work are tested together. Public updates in this repository will focus on broad progress, lessons learned, safe demo material, and simplified examples.
