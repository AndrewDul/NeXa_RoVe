# Videos

This folder is for public-safe videos.

## Public demo video

[NeXa RoVe 26-second demo](nexa-rove-26s-demo.mp4)

A short public-safe video showing the current NeXa RoVe setup.

Suitable videos include:

- short demos
- progress clips
- hardware movement shown safely
- UI clips with private data removed

Do not add:

- raw private recordings
- private conversations
- private code
- terminal logs
- exposed secrets or logs
- diagnostics
- `.env` files
- tokens
- diagnostics screens with sensitive details
- raw audio or transcripts
- user data
- footage that shows personal data or private spaces without review
