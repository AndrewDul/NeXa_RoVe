# Software overview

The software side of NeXa RoVe is mainly built around Python and local-first assistant behaviour.

Public-safe areas include:

- Python runtime work
- local AI direction
- voice interaction
- local speech input and output
- user interface work
- hardware integration
- testing and debugging
- local-first design choices
- safety checks for physical actions

The private implementation is not included here. This repository only contains broad notes and simplified examples that are safe to publish.
